Что делать, если забыл пароль в Linux? #2 09.04.2025

> Если от root, то вводим в терминале:
sudo passwd root

> Если от обычного пользователя, то вводим в терминале:
sudo passwd user
где user - название вашего пользователя

> Если забыты все пароли, то загружаемся с Live CD (ни в коем случае не меняйте пароль root, нельзя оставлять root без пароля).
Подойдёт любой Live CD, где вы можете открыть терминал Linux.
Открываем файл /etc/shadow в текстовом редакторе.

Вот представим строчку:
user:$6$eZ1pQKRK$tGvrlJnV7NU/anEFoxRCluIbe
5kSNE2x2uqRrw62xHVrQPWKKbfYZ0cvdmgxXAcr
GxBngPM7HnrrKxdmNwp0:14371:0:99999:7:::

И поводим до такого вида:
user::14371:0:99999:7:::
То есть мы стираем значения в первых 2 двоеточиях.
